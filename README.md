# machaao-py
An easy to use abstraction module for connecting MACHAAO Platform APIs to your chatbot logic

### Install
```bash
# For Linux/MacOS
pip3 install machaao

# For Windows
pip install machaao
```

### Create new chatbot project
```bash
machaao start <project_name>
```

### Build and Run
```bash
cd <project_name>
python3 chatbot.py
```
